 Le dossier DEVOIR 2 contient mon projet portant sur la geometrie du plan base sur une etude 2D a l'aide des POINTS et des VECTEURS .

 Dans je me suis servie de la programmation modulaire pour ce faire j'ai creer
 
 -un fichier en tete  " structure.h"
 
 -puis j'ai implementer les prototypes des fonctions dans un fichier "structure.cpp"
 qui se sert du fichier en tete.
 
 -j'ai ensuite rediger mon programme dans le  fichier "main.cpp" dans laquelle j'ai declarer mes fonctions .
 
 -j'ai compiler mon programme avec clang++ et j'ai pu avoir le fichier executable "vecteurs.exe"

  l'objectif principale de mon programme est de pouvoir :
   -translater un point
   -determiner l'homethetie d'un point 
   -determiner la rotation d'un point
   -creer un vecteur
   -d'additionner 02 vecteurs 
   -de soustraire 02 vecteurs 
   -calculer le produit scalaire de 02 vecteurs 
   -calciler le determinant de 02 vecteurs 
   -determiner la norme d'un vecteur
   - determiner le LERP ou encore l'interpolation lineaire
     
     Ceux ci l ' objectif principal de mon projet .
